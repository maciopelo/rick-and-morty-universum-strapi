# Rick and Morty Universum Strapi application

Application is used as a backend for [Rick and Morty Universum](https://rick-and-morty-universum.netlify.app/) project.

---

# How to run?

To test locally download [frontend](https://github.com/maciopelo/rick-and-morty-universum)

React APP:

`cd frontend`\
`npm start`

Strapi APP:

`cd backend`\
`npm run strapi develop`
